REST API V1.0

薪資資料串接 API

本文件說明如何透過 API 取得各校區/分部之薪資統計資料，以便與會計系統或 ERP 系統進行帳務同步。

1

基本資訊

傳輸協議

HTTPS

資料格式

JSON

介面路徑 & 方法

GET /api/v1/salary/records
2

認證方式 (Authentication)

Header 名稱 內容 說明
Authorization Bearer {your_api_token} 請替換為您的實際 Token
Content-Type application/json 指定資料格式

請妥善保管您的 API Key，切勿在客戶端前端程式碼中明文寫入。

3

請求參數 (Request Parameters)

參數名稱 類型 必填 範例 說明
month string 是 2026-04 欲查詢的月份 (格式：YYYY-MM)
bunit_id string 否 A01 指定校區編號，不填則回傳所有權限內校區
4

回傳欄位說明 (Response Fields)

根節點物件

欄位名稱 類型 說明
status string 執行狀態，成功為 success
month string 本次查詢的資料月份
data array 各校區資料陣列

data 陣列元素 (校區資訊)

欄位名稱 類型 說明
bunitName string 校區/分部名稱
records array 該校區下的薪資項目明細

records 陣列元素 (薪資明細)

欄位名稱 類型 說明
salaryName string 薪資類別名稱 (如：教職員薪資條、行政人員薪資條)
syncAcct string 對應會計科目編號
payable number 應付薪資金額
syncTime string | null 最後一次同步到外部系統的時間。若為 null 則表示尚未同步過

錯誤代碼 (Error Codes)

HTTP 狀態碼 錯誤訊息 (message) 說明
400 INVALID_MONTH_FORMAT 月份格式錯誤
401 UNAUTHORIZED API Token 無效或已過期
404 DATA_NOT_FOUND 該月份查無薪資資料
500 INTERNAL_SERVER_ERROR 系統內部錯誤
